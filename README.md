# Invitación digital — Starter (vista previa)

Este es un starter para una invitación digital de boda con experiencia "landscape" optimizada para móvil. Es una vista previa local: el botón/formulario RSVP existe pero NO almacena datos en ningún servidor.

Contenido:
- `index.html` — HTML principal con paneles horizontales.
- `styles.css` — Estilos, tono pastel elegante.
- `script.js` — Lógica de navegación horizontal, puntos, formulario preview-only.

Cómo desplegar en GitHub Pages:
1. Crea (o usa) un repositorio en GitHub.
2. Copia estos archivos en la raíz del repositorio.
3. Empuja al branch que tienes configurado para GitHub Pages (por ejemplo `main` o `gh-pages`).
4. En GitHub, ve a Settings → Pages y verifica que el branch correcto esté seleccionado (si ya lo tienes configurado el sitio se publicará automáticamente).
5. Abre la URL de GitHub Pages de tu repo (p. ej. `https://<usuario>.github.io/<repo>/`).

Vista previa local:
- Abre `index.html` en tu navegador (para mejor representación del scroll horizontal prueba en un móvil o ajusta el ancho de la ventana).

Siguientes sugerencias (opcional):
- Integrar backend para guardar RSVPs (Supabase, Netlify Functions, Google Sheets).
- Añadir animaciones adicionales, lightbox en galería, contador regresivo.
- Añadir página /admin para ver RSVPs y exportar CSV.
- Ajustes de contenido, tipografías o paleta según tus fotos reales.

Si quieres, ahora puedo:
- Subir estos archivos en un repo (te puedo generar los archivos listos para copiar/pegar), o
- Añadir almacenamiento básico (por ejemplo guardar en localStorage mientras no haya backend), o
- Preparar el deploy con un pequeño workflow para GitHub Pages.

Dime qué prefieres y seguimos.
